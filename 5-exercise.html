Edit the exercise-app.js file to build a stateful weather card that fetches the weather for Toronto every 15 seconds. Use the OpenWeatherMap API to fetch the weather data. 

http://api.openweathermap.org/data/2.5/weather?id=6167865&appid=5cc1f25759881907aed6171543839b19&units=metric

Reshape the data received from the OpenWeatherMap API to the following: 

{
  temp: data.main.temp,
  description: data.weather[0].description,
  icon: `http://openweathermap.org/img/w/${data.weather[0].icon}.png`,
}
